##########     Backend de Aprende Jugando   #########

Ejecutar los comando de instalacion:
    -pip install virtualenv------Instalaremos un entorno como requisito para gestionar paquetes y programas de python
    
    - virtualen venv------ creamos el entorno virtual y lo nombramos 'venv'
    
    -pip install flask ---- Flask es el framework que usaremos para este proyecto

    -pip install Flask-PyMongo ----- Me permitira tener una conexion con la base de datos, esta se ejecuta en MongoDB

    -pip install flask-cors ----- CORS nos comunicara el backend y el frontend mediantes peticiones HTTP 

Requisitos:
    -Python 3.9.2
    -pip 21.1
    -Flask 1.0.2



Para dar inicio a este API creado ejecuatmosel comando:  python ./src/app.py
